# Algo-Trading-Backtesting
Repositorio utilizado para el Taller de Tesis 1, de la Maestría de Data Mining de la UBA. El proyecto busca comparar los rendimientos obtenidos de seguir las señales de compra y venta de una estrategia de trading basada en la nube de Ichimoku y la rentabilidad que hubiese obtenido las predicciones de una red neuronal.
